Student Performance & Resource Analysis
About the Project

This project is about analyzing student performance using attendance, marks, and resource usage data. The main idea is to understand how students are performing academically and how effectively educational resources are being used. The results are shown through a dashboard for easy understanding.

What This Project Does

Shows overall student performance

Analyzes average attendance and marks

Compares performance between departments

Studies how resources like labs, libraries, and online platforms are used

Identifies areas where improvement is needed

Dashboard Highlights

Total number of students

Average attendance, marks, and resource utilization

Year-wise trend of academic performance

Department-wise comparison

Student performance categories

Data Used

The dataset includes basic academic details such as student information, department, attendance percentage, marks, and resource usage data.

Tools Used

Power BI

DAX for calculations

Data visualization

Conclusion

This analysis helps in understanding student performance patterns and resource usage. It can be useful for improving academic planning, student support, and better use of educational resources.
