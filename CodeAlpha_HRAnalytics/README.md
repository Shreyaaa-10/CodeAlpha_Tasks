# HR Analytics Power BI Project

## Description
This project uses Power BI to analyze HR data. The goal is to understand trends about employees, such as hiring, attrition, job satisfaction, and performance. The dashboard shows the data in a clear and easy-to-read way.

## Purpose
The purpose of this project is to help HR teams and managers make better decisions. Using this dashboard, we can:

- Track how many employees were hired and when  
- See how many employees left the company and the attrition rate  
- Check department-wise performance, satisfaction, and attrition  
- Observe hiring trends over months and years  

This helps make the data easy to understand and the project organized for anyone looking at it.

## Features
- Interactive Power BI dashboard  
- Key metrics like total hires, total attrition, and attrition rate  
- Department-wise analysis for performance and satisfaction  
- Visual charts for trends and forecasting  

## Dataset
The data used in this project includes employee details such as:
- Employee ID  
- Department  
- Job satisfaction  
- Performance rating  
- Hiring and leaving dates  

## Conclusion
This HR Analytics dashboard provides insights about employees in a simple and visual way. It helps organizations understand trends, improve decision-making, and monitor HR performance efficiently.
